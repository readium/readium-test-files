Readium Testing Resources

Revised EPUB Fixed-Layout Test Suite Files

The following files have been cloned from the IDPF EPUB Test Suite. They have been modified to add two enhancements:

  • some Latin verbiage to fill the page up (the original is just a blank page
  • fixed-CSS lines to outline the margins of the page

Note that the CSS is, strictly speaking, not compliant with the EPUB 3.0.1 spec since it uses the CSS "fixed" poisition property. It is hoped we will correct this error (by making fixed legal) in EPUB 3.2, at which point these files can be pushed back into the IDPF repo.

The sources for the test files can be found here.
