Readium Project Resource Files

 

Revised EPUB TestSuite Fixed-Layout Files

The following files have been cloned from the BISG EPUB Test Suite. They have been modified to add two enhancements:

  • some Latin verbiage to fill the page up (the origiinal is just a blank page
  • fixed-CSS lines to outline the margins of the page

Note that the CSS is, strictly speaking, not compliant with the EPUB 3.0.1 spec since it uses the CSS "fixed" poisition property. It is hoped we will correct this error (making fixed illegal) in EPUB 3.2.

The sources for the test files can be found here.

  • EPUBTEST 0201 - FXL Support Tests: Tests for fixed layout support
  • EPUBTEST 0202 - FXL Spreads (both) Tests: Tests for synthetic spreads in both device modes
  • EPUBTEST 0203 - FXL Spreads (landscape) Tests: Tests for synthetic spreads in landscape mode
  • EPUBTEST 0204 - FXL Spreads (none) Tests: Tests for synthetic spreads disabled
  • EPUBTEST 0205 - FXL Spreads (portait) Tests: Tests for synthetic spreads in portrait mode
  • EPUBTEST 0206 - FXL Landscape Orientation Tests: Tests for landscape orientation
  • EPUBTEST 0207 - FXL Portrait Orientation Tests: Tests for portrait orientation
  • EPUBTEST 0208 - FXL Layout Override (reflowable) Tests: Tests for spine overrides of rendition:layout property
  • EPUBTEST 0209 - FXL Layout Override (pre-paginated) Tests: Tests for spine overrides of rendition:layout property
  • EPUBTEST 0210 - FXL Orientation Spine Override Tests: Tests for spine overrides of rendition:orientation property
  • EPUBTEST 0211 - FXL Page Progression Direction Tests: Tests for fixed layout page progression direction
